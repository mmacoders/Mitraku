 Ubah desain halaman Edit Profile yang modern, minimalis, elegan, dan responsif. Gunakan gaya UI/UX premium seperti Apple, Linear, atau Notion.

 Spesifikasi Desain:

 ### 1. Layout Utama

 * Gunakan card container dengan rounded-2xl, shadow halus.
 * Tambahkan efek glassmorphism lembut (background putih/transparan 20–30%).
 * Padding besar (px-6 md:px-10 py-8).
 * Layout harus centered dan responsif.
 * Di atas halaman tampilkan judul:

   * Edit Profil Anda
   * Subtitle kecil: "Perbarui informasi akun dan identitas Anda."

 ### 2. Foto Profil (Avatar Section)

 * Foto profil berbentuk lingkaran diameter 100px.
 * Ada button Upload Foto kecil modern.
 * Tambahkan preview instant setelah foto dipilih.
 * Gunakan style: outline-button, icon kecil, hover subtle.
 * Letakkan di tengah.

 ### 3. Form Input

 Gunakan field dengan style premium:

 * Rounded-xl
 * Border halus `border-gray-300 dark:border-gray-600`
 * Focus ring glow `focus:ring-2 focus:ring-blue-500`
 * Label kecil yang elegan

 Field yang diperlukan:

 * Nama Lengkap
 * Email 
 * Nomor Telepon
 * Alamat / Instansi (mitra only)
 * Password Baru
 * Konfirmasi Password

 Input harus responsif dan rapi (grid 1 kolom di mobile, 2 kolom di desktop).

 ### 4. Notifikasi Validasi

 * Berikan error message kecil berwarna merah dengan icon ⚠️ minimalis.
 * Success message menggunakan green soft dengan icon ✔️.

 ### 5. Tombol CTA

 Letakkan di bawah dalam bar sticky (di mobile) atau di bawah card (desktop).

 * Primary button: Simpan Perubahan

   * Warna biru modern
   * Rounded-xl
   * Shadow soft
   * Hover: sedikit naik (translate-y) dan glow
 * Secondary button: Batalkan

   * Outline, subtle gray

 ### 6. Dark Mode Friendly

 * Background gelap gradasi `from-gray-900 to-gray-800`
 * Card menjadi `bg-white/10` dengan border `border-white/10`
 * Text menjadi `text-gray-200`

 ### 7. Responsiveness

 * Mobile: semua menjadi 1 kolom
 * Tablet: Avatar dan form bersebelahan
 * Desktop: Grid lebih longgar, max width 700–800px

 ### 8. Aesthetic Touch

 * Gunakan icon dari Heroicons / Lucide

 * Tambahkan animasi halus ketika hover/focus

 * Transisi 200–300ms

 * Gunakan spacing nyaman seperti aplikasi modern

 Buatkan kode lengkap untuk tampilan ini dengan Tailwind CSS dan Vue.js (Composition API). Pastikan tampilan sangat bersih, premium, dan seperti aplikasi profesional.
